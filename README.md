# Open Mail

Cliente de email desktop em Tauri v2 + React, iniciado a partir do roadmap em `docs/start`.

## Stack

- Tauri v2
- React 19
- TypeScript + Vite
- Zustand
- TanStack Query
- Tailwind CSS v4
- Vitest + Playwright

## Scripts

- `npm run dev`
- `npm run dev:web`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
